📚 Long Content

Detailed Information
This is an example of a callout with longer content. The component automatically handles text wrapping and maintains proper spacing even with multiple paragraphs or extensive information. The 8px dark blue left border (#1f1f5f) provides strong visual emphasis while the 24px padding ensures comfortable reading space. This design pattern is adapted from the NT Government Web Design System and follows accessibility best practices with WCAG AAA compliant contrast ratios (7:1).

📐 Component Specifications

  • Border: 8px solid #1f1f5f (left)
  • Padding: 24px (Desktop) / 16px (Mobile)
  • Background: #ffffff (white)
  • Text Color: #1f1e27 (dark)
  • Heading: 24px / 700 weight (Desktop), 20px (Mobile)
  • Body: 16px / 400 weight
  • Spacing: 8px gaps, 16px content gap, 1.5rem margin-bottom
  • Accessibility: WCAG AAA compliant (7:1 contrast)
